## 内容
winsocket 编程实现，通过TCP完成客户端与服务端的连接并使用以下三种命令完成交互：
* `dir [path]`
    path 是一个文件夹路径，该命令可查看服务端对应路径下的文件列表
* `upload [path]`
  path 是客户端一个文件路径
* `download [path]`
  path 是服务端的一个文件路径
